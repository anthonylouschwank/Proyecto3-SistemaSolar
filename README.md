# Proyecto Rust

Un programa simple desarrollado en Rust que utiliza controles de teclado para movimiento y cámara.

## Controles

- **Movimiento**: Usa las teclas WASD
  - ⚠️ **Nota**: Las teclas W y S están revertidas
- **Cámara**: Usa las flechas del teclado

## Requisitos

- Tener instalado **Cargo** y **Rust**

## Ejecución

Para ejecutar el programa:

```bash
cargo run
